## 기능 목록

* [ ] 기능선택 기능
* [ ] 과정, 레벨, 미션 선택 기능
* [ ] 페어매칭 기능
* [ ] 같은 레벨에서 같은 매칭이 있는지 확인하는 기능
* [ ] 매칭정보가 있는지 확인하는 기능
* [ ] 매칭정보가 있을시에 재매칭 선택 기능
* [ ] 페어를 조회하는 기능
* [ ] 페어를 초기화 하는 기능

### 예외처리

* [ ] 기능선택시 1, 2, 3, Q이외의 문자는 예외처리
* [ ] 과정, 레벨, 미션 선택시
    * [ ] 과정은 백엔드와 프론트엔드이외의 문자는 예외처리
    * [ ] 레벨은 레벨1 ~ 레벨5 이외의 문자는 예외처리
    * [ ] 미션은 각 미션이외의 문자는 예외처리
* [ ] 재매칭 선택시 네, 아니오 이외의 문자는 예외처리
